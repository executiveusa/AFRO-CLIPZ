# Hero Image Setup

## Required Image

Place your hero image here with the filename:
```
afromations-hero.png
```

## Image Specifications

- **Filename**: `afromations-hero.png`
- **Recommended size**: 1920x1080 or larger
- **Format**: PNG (for transparency) or JPG
- **Style**: Sketch/pencil drawing of Seattle skyline with creative visionary

## How to Add

### Option 1: Direct Upload
1. Save your hero image as `afromations-hero.png`
2. Place it in this folder (`assets/hero/`)
3. Commit and push to GitHub

### Option 2: Git Command
```bash
# From the AFRO-CLIPZ directory
cp /path/to/your/image.png assets/hero/afromations-hero.png
git add assets/hero/afromations-hero.png
git commit -m "Add hero image"
git push
```

## Current Fallback

If the image is not found, the site displays a text-based fallback with:
- "AFROMATIONS" title
- "The AI archive that finds your story" tagline
- "EXPLORE" CTA button

## Image Content Description

The hero image should feature:
- Seattle skyline (Space Needle visible)
- A creative visionary figure (person with afro, futuristic glasses)
- Sketch/pencil drawing aesthetic
- "AFROMATIONS" text at top
- "EXPLORE" button at bottom
